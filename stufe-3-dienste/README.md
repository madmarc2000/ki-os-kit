# Stufe 3 — Eigene Dienste (optional)

Erst hier kommt ein **Server** ins Spiel — und zwar dein eigener.

## Voraussetzung (zusätzlich zu Stufe 1+2)
- Ein **eigener VPS** (beliebiger Anbieter).
- Eine **Domain** + DNS.

## Bausteine (kommen als Nächstes)
- `Caddyfile.beispiel` — Reverse Proxy.
- `docker-compose.beispiel.yml` — Beispiel-Dienst.

> Wird ausgebaut, wenn Stufe 1+2 stehen. Platzhalter.
