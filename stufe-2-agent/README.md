# Stufe 2 — Agent (optional)

Macht aus dem Vault ein aktives System: Ein autonomer Agent liest Aufgaben, führt sie aus
und schreibt Ergebnisse zurück in den Vault.

## Voraussetzung (zusätzlich zu Stufe 1)
- Ein **Always-On-Rechner** (z.B. Mac), der den Agenten laufen lässt.
- Ein **KI-Account** (z.B. Claude) oder API-Key.
- **Immer noch kein VPS nötig.**

## Bausteine (kommen als Nächstes)
- `cowork-aufgaben.md` — Task-Format (`#aufgabe [id]` / `#erledigt`).
- `secrets-konzept.md` — wie Keys NICHT im Vault landen.
